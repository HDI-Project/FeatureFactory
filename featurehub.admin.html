name : str problem_type : str

Dict with additional details about problem. For example, the dict may be {“classification_type” : “multiclass”}.

Absolute path of containing directory of data files for training.

Absolute path of containing directory of data files for testing

List of file paths relative to data_dir; files must be named identically within both data_dir directories.

List of table names, corresponding exactly to files

Name of table that contains the entity variables. Must be found in table_names.

Name of table that contains the pre-processed, featurized, entity variables. Must be found in table_names.
target_table_name : str
Name of table that contains the target variable (label). Must be found in table_names. Table must hold a single column with label values only.

cross_validate(feature)[source]¶

Return score of feature run on dataset sample.

Runs the feature in an isolated environment to extract the feature values. Validates the feature values. Then, builds a model on that one feature, performs cross validation, and returns the score.

discover_features(code_fragment=None)[source]¶

Discover features written by other users.

Print features written by other users, enabling collaboration. A code fragment can be used to filter search results. For each feature, prints feature score and feature code.
